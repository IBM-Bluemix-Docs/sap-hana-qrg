# 2. 為 SAP 安裝準備伺服器 (192 GB)
{: #prepare_256GB}

公用 IP 用於遠端存取，這表示您可以透過 `ssh` 用戶端（例如 Microsoft Windows 上的 PuTTY）連接到伺服器。對您的裝置使用顯示在「裝置清單」（位於**裝置**功能表下）中的公用 IP 位址。也會顯示伺服器的根密碼；務必按一下**顯示密碼**來查看它。
